# JS Snake Game

Famoso `Jogo da Cobrinha`

Desenvolvido com `Javascript` + o poder do `Canvas`

## Fluxo

- Ao abrir o jogo, o mesmo incicia `pausado`
- Pressione o botão `start` para iniciar o *game*
- Para pausar, pressiona o botão `pause`
